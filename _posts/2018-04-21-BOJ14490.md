### 문제 링크
* https://icpc.me/14490

### 문제 출처
* 2017 선린 봄맞이 대회 D번

### 사용 알고리즘
* 유클리드 호제법

### 시간복잡도
* O(log( Max(n, m) ))

### 풀이
<b>최대한 약분한다</b> 라는 것은 <b>최대 공약수로 나눈다</b> 와 같은 의미입니다.<br>
유클리드 호제법을 이용하여 gcd를 구한 뒤 적절히 나누어서 출력하면 됩니다.

### 전체 코드
```cpp
#include <stdio.h>

int gcd(int a, int b){
    if(!b) return a;
    gcd(b, a%b);
}

int main(){
    int g;
    int a, b; scanf("%d:%d", &a, &b);
    g = gcd(a, b);
    printf("%d:%d", a/g, b/g);
}
```
